Tips:

  • the tutorial assumes the web-browser is on the same host as the container(s).
  • curl can be useful when testing connections
  • the tutorial uses YML files, which are picky about indentation, so it might be a good idea to switch from tabs to spaces in vi by running these commands in vi 
    :set tabstop=4
:set expandtab
    or, adding these lines to ~/.vimrc 
    set tabstop=4
set expandtab
