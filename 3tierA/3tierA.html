Lab: 3 Tier Deployment - Windows

Introduction

The combination of Apache-PHP-MySQL can run on almost any hardware and OS. Today we will see what it takes to install our TRS application in a Windows environment.

Windows does not come with software repositories containing these tools the way Linux does. You will have to get Apache and MySQL/MariaDB as downloads.

Part A: Install TRS on a single server

Our main goal is to demonstrate that TRS will run on Windows. Here's what we will build to demonstrate that.

  • No Load balancing is required.
  • You may use any version of Apache Web Server and MySQL/MariaDB
  • You may use any version of Windows you like
