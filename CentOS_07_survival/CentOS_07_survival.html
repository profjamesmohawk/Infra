CentOS 7 Survival Guide

Introduction

This guide is targeted at administrators familiar with CentOS 6. It is intended to get them up and running with a functioning CentOS 7 system quickly. It covers a small set of topics in minimal detail.

Setting the hostname

  • hostname in now set in /etc/hostname

Where did eth0 go?

A new device naming convention is used by udev. It is based on hardwar path not discovery order. For a VirtualBox VM with two interfaces you should see...

CentOS 6 CentOS 7
eth0 enp0s3
eth1 enp0s9

Managing Services

A new system startup daemon, systemd, replaces the startup scripts that we have loved for decades. We will use

  • To set the docker service to start at boot: 
    systemctl enable docker
  • To set the docker service to start at immediately: 
    systemctl start docker

Firewall

The firewall is now managed with firewall-cmd

  • To permanently open port 80/tcp: 
    firewall-cmd --permanent --add-port=80/tcp
firewall-cmd --reload
  • To list open ports: 
    firewall-cmd --list-all
